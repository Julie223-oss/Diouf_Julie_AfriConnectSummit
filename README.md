# AfriConnect Summit 2026

**Auteur** : Julie Diouf  
**Projet** : Examen Technologie Web

## Description
AfriConnect Summit est un site vitrine pour une conférence tech panafricaine fictive.  
L'événement réunit développeurs, entrepreneurs et investisseurs du continent.

## Technologies utilisées
- HTML5 sémantique
- CSS3 (variables CSS, Flexbox, Grid, animations)
- JavaScript vanilla
- Git & GitHub

## Fonctionnalités implémentées
- Dark Mode avec localStorage
- Compte à rebours en temps réel
- Compteurs animés au scroll (Intersection Observer)
- Filtrage dynamique des intervenants
- Onglets interactifs du programme
- Validation complète du formulaire de contact
- Menu hamburger responsive
- Bouton retour en haut
- Année dynamique dans le footer
